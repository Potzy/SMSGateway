Contém as classes modelo que uma SMS pode assumir

Três classes modelo (Recebida/Notificação/Enviada):

  1. Representa a SMS recebida (operação SMS) (ver a classe: {@link pt.gov.dgrm.smsgatebox.models.SMSReceiver})
  2. Representa a notificação de entrega (ver a classe: {@link pt.gov.dgrm.smsgatebox.models.SMSNotification})
  3. Representa a SMS enviada (ver a classe: {@link pt.gov.dgrm.smsgatebox.models.SMSSender})
