Contém as classes principais necessárias no funcionamento do endpoint REST

Este package é responsável por:

  1. Inicar o servidor HTTP (ver a classe: {@link pt.gov.dgrm.smsgatebox.Main})
  2. Escutar de pedidos POST e GET (ver a classe {@link pt.gov.dgrm.smsgatebox.SMSListener})
  3. Filtrar respostas HTTP (ver a classe {@link pt.gov.dgrm.smsgatebox.CORSResponseFilter})
  4. Capturar possíveis exepções (ver a classe {@link pt.gov.dgrm.smsgatebox.DebugExceptionMapper})
